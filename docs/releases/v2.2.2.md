# Astral v2.2.2 - Güncelleme köprüsü ve premium ikon

Tarih: 2026-06-17

v2.2.2, v2.2.1 sonrası görülen otomatik güncelleme asset uyumsuzluğunu ve görünür güncelleme durumunu ele alan hotfix sürümüydü. Bu sürümde geçiş dönemi için ek köprü assetleri denenmiştir; güncel v2.2.29 hattı yalnız `Astral-*` assetleriyle devam eder.

## Değişenler

- Ana ekrandaki güncelleme düğmesi o dönemde **Denetle**, **Denetleniyor**, **Güncel**, **Tekrar dene** ve **Güncelle** durumlarını gösterecek şekilde genişletildi.
- Geçiş dönemi için eski istemcilerin beklediği paket adını karşılayan köprü assetleri üretildi.
- Astral ikon seti eski işarete benzer ama daha premium bir A markasıyla yenilendi.
- Release workflow'u o dönem 8 asset'i birlikte doğruluyordu.

## Güncelleme Notu

Manuel kurulumda GitHub Releases sayfasından `Astral-win-x64.zip` indirilebilir. Otomatik güncelleme için sürümlü `Astral-2.2.2-win-x64.zip` korunur.

Eski istemciler asset adının yanında eski GitHub repo yolunu da doğruladığı için geçişin tam otomatik olması ayrı köprü repo/yayın yüzeyi gerektirir. Bu karar güncel Astral release hattından ayrılmıştır.

## Doğrulama

- `dotnet build Astral.sln --configuration Release`
- `dotnet run --project tests\Astral.Core.Tests --configuration Release`
- `dotnet run --project tests\Astral.Windows.Tests --configuration Release`
- `.\scripts\verify.ps1`
- `gitleaks detect --redact --no-banner --verbose`
- `scripts\build-release.ps1`

## Yayın Asset'leri

Bu tarihsel sürümde Astral assetlerinin yanında geçiş dönemi köprü assetleri de vardı. Güncel yayın modeli için [release denetimi](release-denetimi.md) belgesine bakın.
