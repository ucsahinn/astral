# Astral v2.2.2 - Güncelleme köprüsü ve premium ikon

Tarih: 2026-06-17

v2.2.2, v2.2.1 sonrası görülen otomatik güncelleme asset uyumsuzluğunu ve görünür güncelleme durumunu düzelten hotfix sürümdür. Astral hedef/preset kapsamı korunur; tarayıcı süreçleri WireSock `AllowedApps` kapsamına alınmaz.

## Değişenler

- Ana ekrandaki güncelleme düğmesi artık tamamen kaybolmaz; **Denetle**, **Denetleniyor**, **Güncel**, **Tekrar dene** ve **Güncelle** durumlarını gösterir.
- Eski Discorder 2.1.x istemcileri için `Discorder-<version>-win-x64.zip` köprü paketi üretimi eklendi.
- Köprü paketinde `Discorder.exe`, `Discorder.Updater.exe` ve `discorder.update-manifest.json` bulunur; payload yeni Astral dosyalarını da içerir.
- Köprüden başlatılan `Discorder.exe`, aynı klasörde `Astral.exe` varsa Astral'a yönlenir.
- Astral ikon seti eski işarete benzer ama daha premium bir A markasıyla yenilendi.
- Release workflow'u 8 asset'i birlikte doğrular: Astral sürümlü/sabit ZIP+SHA ve Discorder köprü sürümlü/sabit ZIP+SHA.

## Güncelleme Notu

Manuel kurulumda GitHub Releases sayfasından `Astral-win-x64.zip` indirilebilir. Otomatik güncelleme için sürümlü `Astral-2.2.2-win-x64.zip` korunur.

Eski Discorder otomatik güncelleyicileri, asset adının yanında eski GitHub repo yolunu da doğrular. Bu nedenle `Discorder-2.2.2-win-x64.zip` köprü paketi doğru içerikle üretilse bile eski istemcilerin tamamen otomatik güncellenmesi için GitHub tarafında `ucsahinn/discorder` release yolunun gerçek asset URL'i olarak dönmesi gerekir. Bu repo/yayın yüzeyi ayrı güvenlik onayı gerektiren dış durumdur.
