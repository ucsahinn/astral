# Discorder v2.1.4 - Tooltip polish ve tanılama paneli sadeleşmesi

Yayın tarihi: 2026-06-15

v2.1.4, v2.1.3 sonrası görülen iki küçük arayüz sorununu kapatan patch sürümdür. Tarayıcı modu ve restart iyileştirmeleri korunur; bu sürüm özellikle tooltip görünürlüğünü ve sağ tanılama panelinin gereksiz tekrarını düzeltir.

## Kısa Özet

- Tooltip kutuları artık beyaz/boş görünmez; koyu glass tema, açık metin ve ince cyan border kullanır.
- Sağ tanılama kartındaki ikinci `Rapor hazırla` butonu kaldırıldı.
- Tanılama aksiyonu alt hızlı aksiyon barındaki tek `Tanılama` butonundan yönetilir.
- v2.1.3 ile eklenen sürüm notları ve Discorder yeniden başlatma ikonları korunur.
- Tarayıcı modu için boşluklu browser executable path quoting fix'i korunur.

## Neden Önemli?

Tooltip, disabled/locked kontrollerde kullanıcıya nedenini anlatmak için kullanılıyor. Default WPF tooltip teması video/glass yüzeyde beyaz boş kutu gibi görünebildiği için görünürlük bozuluyordu. v2.1.4 global tooltip template'i ekleyerek tüm buton ve toggle tooltip'lerini aynı koyu Discorder temasıyla gösterir.

Sağ paneldeki tanılama kartı durum bilgisini göstermeye devam eder; rapor oluşturma işi ise alt bardaki mevcut `Tanılama` aksiyonunda kalır. Böylece aynı iş için iki ayrı buton görünmez.

## Değişiklikler

- Global WPF `ToolTip` stili eklendi.
- Tooltip background, foreground, border, shadow ve font ayarları Discorder temasına bağlandı.
- Sağ operasyon panelindeki `Rapor hazırla` butonu pasif `Hazır` rozetiyle değiştirildi.
- Sürüm metadata'sı `2.1.4` olarak güncellendi.
- Uygulama içi release notes popup'ı v2.1.4 metniyle güncellendi.

## Kurulum ve Güncelleme

Manuel kurulumda GitHub Releases sayfasından `Discorder-win-x64.zip` indirilebilir. Sürümlü `Discorder-2.1.4-win-x64.zip` asset'i otomatik güncelleme doğrulaması ve release izlenebilirliği için korunur.

Uygulama içi güncellemede Discorder açılışta denetimi arka planda yapar:

1. Yeni sürüm yoksa update düğmesi görünmez.
2. Yeni sürüm varsa `Güncelle` düğmesi görünür.
3. `Güncelle`, denetlenen aynı sürümü indirir, doğrular ve uygular.
4. Hata olursa mevcut sürüm kullanılabilir kalır.

## Güvenlik ve Güven Sınırı

- Discorder otomatik güncelleme paketini GitHub release yolu, GitHub asset digest, SHA-256 dosyası ve update manifesti eşleşmeden uygulamaz.
- Kod imzalama sertifikası yoksa Windows yayıncı kimliği garanti edilemez; bu sürümde güven sınırı GitHub yayın yetkisi ve doğrulama verileridir.
- WireSock fallback MSI varsa hash, Authenticode, yayıncı ve sürüm doğrulaması geçmeden çalıştırılmaz.
- Tarayıcı modu açıkken WireSock kapsamı process seviyesindedir; browser içindeki yalnızca tek Discord sekmesini Windows/WireSock seviyesinde ayırmak garanti edilemez.

## Bilinen Sınırlar

- Public Authenticode sertifikası olmadığı için SmartScreen veya bazı AV ürünleri uyarı gösterebilir.
- Browser cache, DoH, IPv6 veya mevcut açık bağlantılar nedeniyle tarayıcı modu değişikliği yeni oturum gerektirebilir.
- GitHub erişimi olmayan ağlarda update check başarısız olabilir; uygulama mevcut sürümle çalışmaya devam eder.
