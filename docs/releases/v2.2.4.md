# Astral v2.2.4 - Inline hedef kartları ve güncelleme hotfix

Tarih: 2026-06-17

v2.2.4, Astral'ın ürün yüzeyini tamamen Astral adına sabitleyen, hedef seçimini ana ekrana taşıyan ve otomatik güncelleme görünürlüğünü düzelten hotfix sürümüdür.

## Değişenler

- Hedef seçimi artık ayrı pencere açmadan ana ekrandaki kartlarla yapılır.
- Hazır hedef listesi Discord, Roblox, Wattpad, Bigo Live, Azar, Tango, LiVU, IMVU ve Blogspot ile sınırlandı.
- Desteklenmeyen eski özel hedef yüzeyi kaldırıldı; eski ayar alanları migration sırasında route planına taşınmaz.
- Güncelleme düğmesi yalnız yeni sürüm bulunduğunda görünür hale getirildi.
- Yeni sürüm bulunursa tek **Güncelle** düğmesi, arka planda denetlenmiş aynı paketi indirip doğrular.
- Access-lock hataları hangi aşamada oluştuğunu gösterecek şekilde tanılama ve kullanıcı mesajına yansır.
- Astral ana release hattı yalnız `Astral-*` ZIP ve SHA assetlerini üretir.

## Güncelleme Notu

Manuel kurulumda GitHub Releases sayfasından `Astral-win-x64.zip` indirilebilir. Otomatik güncelleme için sürümlü `Astral-2.2.4-win-x64.zip` korunur.

Eski istemci geçişi ana Astral release assetleriyle yapılmaz. Geçiş dönemi için ayrı köprü repo/yayın yüzeyi kullanılabilir; Astral'ın güncel updater yolu `Astral-*` asset adlarına bağlı kalır.

Kod imzalama sertifikası bu build ortamında yapılandırılmadıysa paket imzasızdır. Güncelleme güven sınırı GitHub release yolu, asset digest, SHA-256 dosyası ve manifest eşleşmesidir.

## Doğrulama

- `dotnet build Astral.sln --configuration Release --disable-build-servers`
- `dotnet run --project tests\Astral.Core.Tests --configuration Release`
- `dotnet run --project tests\Astral.Windows.Tests --configuration Release`
- `powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify.ps1`
- `gitleaks detect --redact --no-banner --verbose`
- `scripts\build-release.ps1`
