# Astral VPN v2.2.24 - Hedef presetleri ve canlı proof hotfix

> Not: v2.2.24 GitHub asset'i yayınlandıktan sonra arka plan video paketinin yerel doğrulanan yeni video build'iyle eşleşmediği görüldü. Aynı sürüm numarasını yeniden kullanmak otomatik güncellemeyi tetiklemeyeceği için video ve paket hotfix'i v2.2.25 ile üstlenildi.

v2.2.24, hedef kapsamını 16 hazır preset'e genişletir ve canlı bağlantı doğrulamasını seçilen hedef listesine göre sertleştirir. Presetler yalnız route kapsamını tanımlar; erişim durumu zamanla değişebilir.

## Değişiklikler

- Ürün yüzeyi `Astral VPN` adıyla güncellendi; portable dosya, ZIP ve updater hattı mevcut `Astral-*` adlarını korur.
- Roblox, Radio Garden, DW, VOA, Ekşi Sözlük, Grok, Imgur ve Pastebin hedefleri eklendi.
- Her hedef için `launchUrl` metadata'sı eklendi. Bağlantı açıkken seçili hedef kartına tıklamak ilgili web girişini varsayılan tarayıcıda açar.
- Hedef kartlarındaki seçim tiki kaldırıldı. Seçili hedefler yeşil/cyan çerçeveyle, seçili olmayan hedefler pasif opaklıkla gösterilir.
- Sağ hedef paneli 16 hedefi sabit 1280x720 pencere içinde 8+8 kompakt grid olarak gösterir.
- Canlı smoke script'i `-TargetIds` parametresi aldı ve `webProxyProof` verified/count/failed-target alanlarını critical check haline getirdi.
- Canlı smoke script'i tek string halinde gelen virgüllü hedef listesini de böler; tekli, web-only subset ve app+web subset seçimleri release öncesi ayrı kanıtlanır.
- Ayar okuyucu `TargetSelection` alanını case-insensitive okur. Eski veya elle düzenlenmiş `targetSelection.selectedTargetIds` ayarları artık güvenli default'a düşmeden seçilen hedefleri korur.
- Hedef ikon resource listesi 16 hedefe genişletildi; release build'de letter fallback'e düşmemesi test edilir.

## Güvenlik Sınırı

- Web hedeflerinde genel tarayıcı süreçleri WireSock `AllowedApps` kapsamına alınmaz.
- Seçili domainler PAC üzerinden yalnız `Astral.WebProxy.exe` sürecine yönlenir; diğer domainler `DIRECT` kalır.
- HTTPS içeriği çözülmez, sertifika kurulmaz ve TLS MITM yapılmaz.

## Doğrulama

- `dotnet build Astral.sln --configuration Release`
- `dotnet run --project tests\Astral.Core.Tests --configuration Release`
- `dotnet run --project tests\Astral.Windows.Tests --configuration Release`
- `powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify.ps1 -ArtifactsPath artifacts\verify-v2.2.24`
