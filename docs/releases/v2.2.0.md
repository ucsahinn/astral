# Astral v2.2.0 - Hedef/preset tabanlı dar kapsam

v2.2.0, Astral'ı tek hedefli bağlantı aracından seçilebilir hedef/preset tabanlı dar kapsamlı bağlantı yöneticisine taşır.

## Değişenler

- Hedef Merkezi eklendi.
- Discord, Roblox, Wattpad, Bigo Live, Azar, Tango, LiVU, IMVU, Blogspot, Özel EXE ve Özel Domain presetleri eklendi.
- Ana ekrandaki kapsam kartı “TÜNEL KAPSAMI” olarak yenilendi.
- Görünür eski web kapsam toggle'ı kaldırıldı.
- TargetRegistry, TargetSelectionStore, TargetScopeResolver ve RoutingPlan modeli eklendi.
- `BuildScopedProfile` ile profil üretimi generic hedef planına taşındı.
- `Astral.WebProxy.exe` eklendi.
- Web hedeflerinde genel tarayıcı süreçlerinin WireSock `AllowedApps` kapsamına girmesi engellendi.
- PAC üretimi seçili domainlerde PROXY, diğer domainlerde DIRECT döndürür.
- Custom EXE/domain validation testleri eklendi.

## Güvenlik Notları

- HTTPS için TLS MITM/decryption yoktur.
- Sertifika kurulmaz.
- Proxy allowlist dışı hostlara izin vermez.
- Browser exe'leri web hedefleri için tünel kapsamına alınmaz.
- PAC/proxy state'i bağlantı kapanınca, hata alınca ve uygulama kapanınca restore edilir.
