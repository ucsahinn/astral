# Astral v2.2.1 - Seçili hedef kapsamı tamamlama

Tarih: 2026-06-17

v2.2.1, v2.2.0 hedef/preset mimarisini tamamlayan patch sürümdür. Web-only hedeflerde Discord sürecine dokunulmaz; seçili olmayan uygulama ve domainler tünel planına girmez.

## Değişenler

- Web-only hedef seçiliyken bağlantı akışı Discord yenileme/kapama yoluna girmez.
- Bağlantı denetleyicisi Discord sürecini yalnızca Discord hedefi seçiliyse yönetir.
- Ana UI tooltip, ilerleme metni, profil temizleme metni ve GitHub şablonları seçili hedef kapsamına göre güncellendi.
- Release workflow idempotency kontrolü düzeltildi: mevcut release asset'i varsa lokal ZIP byte eşitliği yerine asset digest, SHA dosyası ve versioned/stable hash eşleşmesi doğrulanır.
- Eski release notlarındaki güncel ürünü yanıltabilecek açılış cümleleri tarihsel bağlamla düzeltildi.

## Güvenlik Notları

- Genel tarayıcı süreçleri WireSock `AllowedApps` kapsamına eklenmez.
- Web hedefleri yalnız `Astral.WebProxy.exe` ve seçili domain allowlist'i üzerinden kapsama girer.
- HTTPS içerik çözme, TLS MITM veya sertifika kurulumu yoktur.
- Custom EXE ve Custom Domain doğrulamaları değişmeden korunur.
