# Astral v2.2.3 - Hedef Merkezi ve bağlantı kilidi hotfix

Tarih: 2026-06-17

v2.2.3, v2.2.2 üstüne gelen UI ve bağlantı dayanıklılığı hotfix sürümüdür. Astral hedef/preset kapsamı korunur; tarayıcı süreçleri WireSock `AllowedApps` kapsamına alınmaz.

## Değişenler

- Hedef Merkezi kategori paneli kaldırılarak daha geniş kart grid'i, premium servis işaretleri, net seçili tik durumu ve taşma kontrollü arama akışı eklendi.
- Hedef kartları artık ad, ikon anahtarı, domain, exe ipucu ve alias alanlarıyla filtrelenir.
- Boş arama sonucu ve dar pencere düzeni sabit genişlik taşması üretmeyecek şekilde düzeltildi.
- WireSock hazırlık denetimi `wt*` + `WireGuard Tunnel` adaptörlerini tanır; ilgisiz `wg-*` adaptör adları kapsama alınmaz.
- Windows Firewall bağlantı kilidi kapanışında DNS flush komutu boş native çıkış koduyla başarısız görünürse tünel akışı kırılmaz.
- Hedef Merkezi render testi kategori panelinin kaldırıldığını, hedef kartlarını, boş arama sonucunu ve ekran görüntüsü üretimini doğrular.

## Güncelleme Notu

Manuel kurulumda GitHub Releases sayfasından `Astral-win-x64.zip` indirilebilir. Otomatik güncelleme için sürümlü `Astral-2.2.3-win-x64.zip` korunur.

Eski Discorder otomatik güncelleyicileri için release içinde `Discorder-2.2.3-win-x64.zip` köprü paketi de yayınlanır. Bu paket `Discorder.exe`, `Discorder.Updater.exe` ve `discorder.update-manifest.json` içerir; payload yeni Astral dosyalarıyla aynı kod tabanını taşır.

Kod imzalama sertifikası bu build ortamında yapılandırılmadığı için paket imzasızdır. Güncelleme güven sınırı GitHub release yolu, asset digest, SHA-256 dosyası ve manifest eşleşmesidir.
