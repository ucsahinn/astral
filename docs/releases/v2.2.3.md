# Astral v2.2.3 - Hedef kartları ve bağlantı kilidi hotfix

Tarih: 2026-06-17

v2.2.3, v2.2.2 üstüne gelen UI ve bağlantı dayanıklılığı hotfix sürümüydü. Astral hedef/preset kapsamı korunur; tarayıcı süreçleri WireSock `AllowedApps` kapsamına alınmaz.

## Değişenler

- Hedef seçim yüzeyinde daha geniş kart grid'i, premium servis işaretleri, net seçili tik durumu ve taşma kontrollü arama akışı denendi.
- Hedef kartları ad, ikon anahtarı, domain, exe ipucu ve alias alanlarıyla filtreleniyordu.
- Boş arama sonucu ve dar pencere düzeni sabit genişlik taşması üretmeyecek şekilde düzeltildi.
- WireSock hazırlık denetimi `wt*` + `WireGuard Tunnel` adaptörlerini tanır; ilgisiz `wg-*` adaptör adları kapsama alınmaz.
- Windows Firewall bağlantı kilidi kapanışında DNS flush komutu boş native çıkış koduyla başarısız görünürse tünel akışı kırılmaz.

## Güncelleme Notu

Manuel kurulumda GitHub Releases sayfasından `Astral-win-x64.zip` indirilebilir. Otomatik güncelleme için sürümlü `Astral-2.2.3-win-x64.zip` korunur.

Bu tarihsel sürümde geçiş dönemi köprü assetleri de yayınlanmıştır. Güncel v2.2.24 hattı bu köprü paketlerini ana Astral release'inde üretmez.

Kod imzalama sertifikası bu build ortamında yapılandırılmadıysa paket imzasızdır. Güncelleme güven sınırı GitHub release yolu, asset digest, SHA-256 dosyası ve manifest eşleşmesidir.
