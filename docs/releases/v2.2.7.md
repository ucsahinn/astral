# Astral v2.2.7 - WebProxy port fallback ve hedef kartları hotfix

Tarih: 2026-06-17

v2.2.7, v2.2.6 tanılama paketlerinde görülen yerel WebProxy port çakışmasını ve hedef kartı görsel okunurluğunu hedefleyen hotfix sürümüdür.

## Değişenler

- Varsayılan yerel proxy portu kullanımda ise Astral bu oturum için yedek loopback port seçer.
- PAC dosyası seçilen gerçek proxy portuna göre yazılır; seçili olmayan domainler yine DIRECT kalır.
- `Astral.WebProxy` başlatılamaz veya hemen kapanırsa PAC uygulanmadan hata verilir.
- WebProxy port çakışması artık yakalanır ve tanılama loguna okunabilir hata koduyla düşer.
- Hedef kartları isim, kapsam etiketi ve marka renkli işaretlerle daha okunur hale getirildi.
- Bağlantı açıkken kilitli hedef kartlarının seçili tikleri soluk griye düşmeden görünür kalır.
- Footer'daki GitHub ikonu ve hedef kartı seçili işareti yüksek DPI'da kırpılmayacak şekilde düzenlendi.
- Ana durum metinlerinde teknik motor adı yerine Astral tünel motoru dili kullanılır.

## Güvenlik ve Kapsam

- Genel tarayıcı süreçleri WireSock `AllowedApps` kapsamına alınmaz.
- Web hedefleri için yalnızca seçili domainler PAC/WebProxy kapsamından geçer; TLS içeriği çözülmez.
- Proxy port fallback yalnız loopback üzerinde çalışır ve sistem geneli VPN davranışı oluşturmaz.
- Üçüncü taraf resmi marka dosyaları bu sürüme gömülmedi; hedef kartları lisans riski oluşturmayan marka renkli fallback marklarla kalır.

## Güncelleme Notu

Manuel kurulumda GitHub Releases sayfasından `Astral-win-x64.zip` indirilebilir. Otomatik güncelleme için sürümlü `Astral-2.2.7-win-x64.zip` korunur.

Kod imzalama sertifikası bu build ortamında yapılandırılmadıysa paket imzasızdır. Güncelleme güven sınırı GitHub release yolu, asset digest, SHA-256 dosyası ve manifest eşleşmesidir.
