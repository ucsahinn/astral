# Astral Public Discovery Metinleri

Bu sayfa README, GitHub açıklaması, release notu ve kısa ürün tanıtımı için tekrar
kullanılabilecek doğal metinleri toplar. Amaç arama motorlarına ve GitHub
kullanıcılarına ürünün ne olduğunu net anlatmaktır; spam veya resmi ortaklık
iddiası yoktur.

## Kısa Başlık Adayı

Astral - Windows için seçili hedef bağlantı yöneticisi

## Meta Açıklama Adayı

Astral, Windows'ta yalnız seçilen uygulama veya web hedeflerini dar kapsamlı
tünel planına alan portable masaüstü uygulamasıdır; diğer trafik normal hatta
kalır.

## Tek Cümlelik Özet

Astral, Discord, Roblox, Wattpad, Blogspot ve özel exe/domain hedefleri gibi
seçili kapsamları tek ekrandan yönetmeye yardımcı olan portable Windows
uygulamasıdır.

## Özellik Mesajları

- Hedef Merkezi üzerinden hazır presetler veya özel exe/domain hedefleri seçilir.
- Uygulama hedefleri WireSock `AllowedApps` kapsamına yalnız kendi exe'leriyle girer.
- Web hedeflerinde genel tarayıcı süreçleri profile eklenmez; yalnız seçili
  domainler `Astral.WebProxy` allowlist'iyle taşınır.
- Portable ZIP olarak çalışır; kullanıcı çıkarılmış klasörü kontrol eder.

## Güven Mesajları

- Tüm PC'yi, tüm tarayıcıyı veya sistem genelini VPN'e almayı amaçlamaz.
- HTTPS içeriğini çözmez, sertifika kurmaz ve web trafiğini okumaz.
- Uygulama içi otomatik güncelleme paketlerini GitHub release yolu, asset
  digest, SHA-256 ve manifest kontrolleriyle doğrular.
- Discord, Roblox, Wattpad, Cloudflare veya WireSock tarafından üretilen resmi
  bir ürün olduğunu iddia etmez.

## Kurulum ve Güncelleme Mesajları

- GitHub Releases üzerinden en güncel ZIP arşivini indirin ve yayınlanan SHA-256
  dosyasıyla kontrol edin.
- ZIP'i kalıcı bir klasöre çıkarıp `Astral.exe` dosyasını çalıştırın.
- Astral açılışta güncellemeyi sessizce denetler; yeni sürüm varsa tek
  `Güncelle` düğmesiyle yüklenir.

## Kapsanan Kullanıcı Soruları

README ve dokümanlar şu kullanıcı sorularını doğal biçimde yanıtlayacak şekilde
yazılmıştır:

- Astral nedir ve Windows'ta ne işe yarar?
- Hedef merkezi nasıl kullanılır?
- Seçilmeyen uygulama veya domain trafiği neden normal bağlantıda kalır?
- Portable ZIP nasıl indirilir, çalıştırılır ve güncellenir?
- Uygulama içi güncelleme neden yalnızca gerçek yeni sürüm bulunduğunda düğme gösterir?
- Astral sistem DNS'ini, tüm cihaz trafiğini veya kullanıcı hesaplarını değiştirir mi?
- Tanılama paketi hangi durumlarda hazırlanır?
